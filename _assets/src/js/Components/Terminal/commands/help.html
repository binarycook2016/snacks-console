> welcome to snacks@console help section, these are the available internal commands that you can use :

  • -settings : open a settings UI where you can configure the behaviors of the console
  • -documentation : open a page for in depth documentation
  • -bookmarks : shows the bookmarked queries
  • -clearrecent : clears the recent queries
  • -clear : clears the console
  • -about : about snacks console
